# Case_module_4
ТРЕКЕР ПРОЧИТАННЫХ КНИГ
Программа предназначена для работы с данными книгами

Чтобы запустить программу необходимо запустить python main.py

Описание работы программы:
В программе реализованы 8 функций:
Функция load_books загружает данные из json-файла,
Функция save_books сохраняет данные в json-файле,
Функция add_book добавляет книгу в список книг,
Функция list_books показывает список книг,
Функция stats_books показывает среднюю оценку книг в списке,
Функция stats_author показывает сколько книг содержится у каждого автора,
Функция delete_book удаляет книгу по ее индексу в списке,
Функция main - главная функция программы, которая вызывает каждую функцию по ее номеру ввода.
